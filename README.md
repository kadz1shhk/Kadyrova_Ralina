Currency Converter – Конвертер валют

Автор : КАДЫРОВА РАЛИНА

 Описание программы
- Графическое приложение (GUI) для конвертации валют
- Использует актуальные курсы с внешнего API (exchangerate-api.com)
- Поддерживает 9 валют: USD, EUR, RUB, GBP, JPY, CNY, BYN, KZT, UAH
- Сохраняет историю всех конвертаций в файл `history.json`
- Позволяет просматривать историю и очищать её одной кнопкой
- Проверяет корректность ввода: сумма должна быть положительным числом

Как запустить приложение :
python main.py

Примеры использования :
             Действие                                                    Результат
Ввести 100, выбрать USD → EUR, нажать «Конвертировать»         100.00 USD = 85.00 EUR (курс актуальный)
Ввести 0 или -50 или abc                                    Ошибка: «Сумма должна быть положительным числом!»
Конвертировать USD → USD                                    Результат равен введённой сумме (без запроса API)
Нажать «Очистить историю»                                   Все записи удаляются из таблицы и из history.json
Закрыть и снова открыть программу                                  История загружается автоматически
